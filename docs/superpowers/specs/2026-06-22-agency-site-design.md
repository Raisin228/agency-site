# Agency Site — Design Spec
**Date:** 2026-06-22
**Status:** Approved (v2 — agency-focused, bilingual, founder blog)

---

## 1. Контекст и цель

Сайт AI-агентства. Аудитория — предприниматели и малый/средний бизнес (приходят по прямой ссылке).

**Цель:** показать кейсы (выполненные проекты для заказчиков), объяснить какие проблемы мы закрываем для предпринимателей, вызвать доверие, получить обращение в Telegram или email.

**Сценарий:** клиент заходит → видит проблемы, которые мы решаем → видит реальные кейсы → видит основателя → пишет.

**Этап:** wireframe с заглушками. Контент (кейсы, фото, отзывы, название агентства) заполнит владелец позже. Название бренда TBD.

---

## 2. Язык и локализация

Сайт двуязычный:

| Версия | Путь | Описание |
|--------|------|----------|
| Русская | `/ru/` | Основная версия |
| Английская | `/en/` | Международная версия |
| Корень | `/index.html` | JS-редирект по языку браузера → `/ru/` или `/en/` |

- Переключатель языка в навигации (RU / EN)
- Тексты в каждой HTML-версии прописаны вручную (без JS i18n)
- `<html lang="ru">` и `<html lang="en">` соответственно

---

## 3. Референс и принципы

**Референс:** galt.agency, galt.academy — минимализм, много воздуха, крупная типографика.

**Принципы дизайна:**
- Белый фон, чёрный текст, много пространства
- Язык агентства: "мы", "наши клиенты", "наши кейсы"
- Красота в деталях: едва заметные анимации, не перегружать
- Контент-первый: анимации поддерживают, не отвлекают
- Заглушки сейчас — структура легко наполняется

---

## 4. Стек

| Компонент | Решение |
|-----------|---------|
| Разметка | HTML5 (семантическая) |
| Стили | CSS3 (custom properties, без фреймворков) |
| Анимации | Vanilla JS + CSS transitions/transforms |
| Деплой | GitHub Pages (статика) |
| Сборщик | Нет |
| Зависимости | Нет |

## 7. Главная страница — секции

### 7.1 Nav
```
[AGENCY NAME TBD]    Projects  Blog    RU|EN    [→ Contact]
```
- Логотип / название слева
- Ссылки по центру (Projects, Blog)
- Справа: переключатель языка + CTA-кнопка (outlined, hover: fill)
- При скролле: `backdrop-filter: blur(12px)` + тонкая нижняя граница появляется
- Ссылки: animated underline при hover (`::after`, `width: 0 → 100%`, 0.3s)

### 7.2 Hero
```
Строим              [Подзаголовок: кто мы и для кого.
Автоматизируем      Заглушка 2-3 предложения.]
Масштабируем
                    [→ Смотреть кейсы]
```
- Слова меняются автоматически: JS интервал + CSS `opacity` + `translateY`
- Крупный шрифт (`clamp(48px, 8vw, 120px)`), тонкий `font-weight: 300`
- CTA → ссылка на раздел #projects или projects.html
- RU слова: Строим / Автоматизируем / Масштабируем
- EN words: We Build / We Automate / We Scale

### 7.3 Services (проблемы, которые мы закрываем)
```
┌──────────────────────────────────────────────────┐
│  Автоматизация рутины                      →     │
│  [описание выезжает при click]                   │
├──────────────────────────────────────────────────┤
│  AI-агенты и системы                       →     │
├──────────────────────────────────────────────────┤
│  Backend-разработка                        →     │
├──────────────────────────────────────────────────┤
│  Стратегия и консалтинг                    →     │
└──────────────────────────────────────────────────┘
```
- Accordion: клик → цветная обводка (`#2563EB`) + описание выезжает снизу
  (`max-height: 0 → 120px`, `opacity: 0 → 1`, 0.4s ease)
- Стрелка `→` вращается в `↓` при раскрытии (`rotate(90deg)`, 0.3s)
- Один блок открыт одновременно
- Описание: конкретная проблема, которую мы закрываем (заглушки)

### 7.4 Projects Preview
```
Наши работы ──────────────────────── Все кейсы →

┌────────────────┐  ┌────────────────┐  ┌────────────────┐
│   [placeholder]│  │   [placeholder]│  │   [placeholder]│
│                │  │                │  │                │
│ Название       │  │ Название       │  │ Название       │
│ AI Agent · Py  │  │ Backend · API  │  │ Agent · Auto   │
└────────────────┘  └────────────────┘  └────────────────┘
```
- 3 featured кейса на главной
- **Cursor glow:** `mousemove` → CSS custom properties `--mouse-x`, `--mouse-y` → `radial-gradient`
- Hover: лёгкое затемнение изображения (`img` scale 1.03)
- "Все кейсы →" → projects.html

### 7.5 Founder (trust block, не биография)
```
┌──────────────┐   [Founder Name TBD]
│              │   Основатель и lead-разработчик
│  [фото]      │
│              │   [1-2 предложения: миссия агентства.
└──────────────┘    Заглушка.]

                   → Читать блог
```
- Минималистичный блок: фото слева, текст справа
- Нет статистик, нет гистограмм — только личность за агентством
- Ссылка "Читать блог" → blog.html

### 7.6 Testimonials
```
"Заглушка: автоматизировали весь процесс, экономим 15 часов в неделю."
— [Имя], [Должность] в [Компания]

"Заглушка: AI-агент обрабатывает 80% входящих запросов без участия команды."
— [Имя], [Должность] в [Компания]
```
- 2 цитаты, без карусели
- Крупные кавычки серого цвета (`:before` pseudo-element)
- Имя/роль: мелкий, серый

### 7.7 CTA
```
Готовы автоматизировать
свой бизнес?

[→ Telegram]          [→ Email]
```
- Крупный заголовок, по центру
- Две кнопки: filled + outlined
- **Text flip при hover:** CSS `perspective` + `rotateX(-90deg → 0)` (0.35s)

### 7.8 Footer
```
[AGENCY NAME TBD]         tg · linkedin · email         RU | EN
© 2026
```
- Тонкая верхняя граница
- Соцсети: hover → animated underline

- 6 карточек-заглушек

## 9. Страница блога (blog.html)

- Список статей основателя: доверие к бренду, не биография
- Карточка: дата, заголовок, превью 2-3 предложения
- 3 карточки-заглушки
- Без комментариев, без CMS — статический HTML
- Статьи не линкуются (нет отдельных страниц статей)

## 10. Микро-взаимодействия

| Эффект | Где | Механика |
|--------|-----|----------|
| Cursor glow | Project cards | JS `mousemove` → CSS `radial-gradient` via custom props |
| Hover border | Service blocks | CSS `border-color` transition + 0.3s |
| Text flip | CTA кнопки | CSS `perspective` + `rotateX(90deg)` + 0.35s |
| Hover reveal | Service description | CSS `max-height: 0 → 120px`, `opacity: 0 → 1` |
| Animated underline | Nav, footer links | CSS `::after`, `width: 0 → 100%` 0.3s |
| Word rotate | Hero heading | JS interval + CSS `opacity` + `translateY` 0.4s |
| Nav blur | Nav при скролле | JS `scroll` event + `backdrop-filter: blur(12px)` |

## 12. Адаптивность

| Breakpoint | Изменения |
|------------|-----------|
| < 768px | Projects grid → 1 колонка; Founder → фото сверху, текст снизу; nav links скрыты |
| < 480px | Hero font уменьшается; Nav → бургер-меню; `--section-gap: 80px` |

---

## 13. Что НЕ входит в скоуп

- Тёмная тема
- CMS или бэкенд
- Форма обратной связи (только ссылки tg/email)
- Авторизация, БД
- Полноценные страницы статей в блоге (только список с превью)
